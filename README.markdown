# `rw` - generate random words

Simple command line random word generator.

## Installation

Simply run

    pip install rw

## Usage

Usage:

    rw [--dict=<wordfile>] <numberofwords>
    rw (-h | --help)
    rw --version

Options:

    -h --help             Show help screen.
    --version             Show version.
    --dict=<wordfile>     Use given word file (optional).
